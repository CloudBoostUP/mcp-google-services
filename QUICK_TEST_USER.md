# Quick Guide: Add Test User (Polish UI)

## Kroki po polsku (Steps in Polish)

1. **Kliknij "Odbiorcy" w lewym menu** (Click "Audience" in left menu)
2. **Przewiń do sekcji "Użytkownicy testowi"** (Scroll to "Test users" section)
3. **Kliknij "+ DODAJ UŻYTKOWNIKÓW"** (Click "+ ADD USERS")
4. **Wpisz swój adres Gmail** (Enter your Gmail address)
5. **Kliknij "DODAJ"** (Click "ADD")

## After Adding Test User

Go back to Cursor and try: **"List my Gmail labels"**

The OAuth flow should work now! ✅
